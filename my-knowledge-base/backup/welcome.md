# 欢迎

欢迎来到我的个人知识库！

## 这是什么？
这是一个完全**静态**的知识库网站，由以下技术驱动：
- **Next.js 14** - React框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式
- **GitHub Pages** - 免费托管

## 如何工作？
1. 我在 `content/posts` 目录下用 Markdown 写文章。
2. 将代码推送到 GitHub。
3. GitHub Actions 会自动构建并发布网站。
